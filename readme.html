Setup Instructions

Prerequisites

  • Install Node.js (v14 or above).
  • Ensure MongoDB is installed and running.

Installation

  1. Clone the repository: 
    git clone https://github.com/AkshayMobileApplicationEngineer/BackendPoint.git
cd BackendPoint
  2. Install dependencies: 
    npm install
  3. Set up environment variables: 
    Create a .env file in the root directory.
Add the following values:
MONGO_URI=your_mongodb_connection_uri
PORT=your_port_number
  4. Start the server: 
    npm start

API Endpoints

Method Endpoint Description
GET /api/status Checks server status
POST /api/login Handles user authentication
GET /api/devices Fetches IoT device list

Deployment

To deploy the backend to a cloud provider like Heroku or AWS, follow the steps in their documentation. Make sure you configure environment variables for MongoDB, Firebase authentication, and cloud storage for seamless integration.
